Dust Collection

Magnetic Port Snapping

Every duct piece, elbow, Y-joint, T-joint, and blast gate has invisible "ports" at the points where it's meant to connect to another piece. When you drag one of these near a compatible open port, it automatically snaps into place — sliding and rotating itself so the two pieces line up flush and facing each other. You don't have to nudge things pixel-by-pixel to get a clean-looking run.

Dragged within range — snapped

How to connect two pieces

  1. Place your first piece (usually starting from the dust collector) anywhere on the grid.
  2. Drag a second piece — a duct segment, elbow, Y-joint, T-joint, or blast gate — near one end of the first piece.
  3. You don't need to be exact. Once you're reasonably close to the open end, the piece snaps: it repositions and rotates itself so its port lines up exactly with the other piece's port.
  4. Keep going, piece by piece, working outward from the collector toward each tool.
Only matching duct sizes snap to each other. A 4" piece will only snap to another 4" piece, and a 6" piece only to another 6". If a piece won't snap the way you expect, double-check both pieces are the same diameter — that's the most common reason.

How many ports does each piece have?

Ports are the connection points a piece is trying to snap. Some pieces only have two (a straight through-connection); fittings that split the airflow have three.

Straight Duct
2 ports (both ends)
Blast Gate
2 ports (both ends)
90° Elbow
2 ports (each leg)
Y-Joint
3 ports
T-Joint
3 ports
Duct Drop
2 ports (both ends)

Blast gates and "open" ends

A blast gate is a valve, not a plain pass-through — in a real shop, one side connects back to the main trunk and the other side sits right at the tool it serves. That tool-facing side is supposed to end there; it isn't meant to connect to anything else, since tools themselves aren't part of the duct network in this planner.

That's why Check Duct Connections (in the Sizing Calculator panel) only flags a blast gate if it isn't hooked into the duct network on either side. A gate with one side properly connected and the other terminating at a tool is exactly right and won't be flagged.

Duct Drops (going from overhead down to a tool)

This grid is a top-down 2D view, so it can't show a pipe actually descending from an overhead trunk to a tool at floor height — every duct piece just floats at the same fixed mount elevation. A Duct Drop is how a vertical run is represented instead: place one where the overhead branch reaches a tool (typically right behind the blast gate), and it snaps into the run exactly like a duct segment does.

In 2D it shows as a plain bullseye icon rather than a pipe shape, since "direction" doesn't mean anything for a vertical connector in a top-down view. It counts as an elbow-equivalent in the sizing calculator (same friction-loss penalty, just for a change in elevation instead of heading), and it's the one dust-collection piece that renders as a real vertical pipe in the 3D Shop Viewer.

Tip: Placing a tool with Attach to dust collection? checked (in the Workshop Layout Planner catalog) automatically builds the blast gate, drop, and duct run for you — drops are inserted automatically, you don't have to add them by hand unless you're routing manually.

Verifying and using your run

  • 🔍 Check Duct Connections — scans every placed piece and marks any genuinely open (unconnected) duct end in red on the grid, so you can spot a gap in the run at a glance.
  • ⚡ Auto-Fill Branches from Layout — traces the connected run from your dust collector out to each blast gate and fills in the branch length and elbow count for you in the sizing calculator below, instead of measuring by hand.
Tip: Build outward from the collector one connected piece at a time rather than placing everything first and connecting it after — it's much easier to see exactly what's snapping to what as you go.
